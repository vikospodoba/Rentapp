Описание проекта

RentApp — это веб‑приложение для управления арендой автомобилей, разработанное в рамках курсового проекта по дисциплине «Современные технологии программирования». Система автоматизирует процессы подбора автомобиля, оформления аренды, учёта платежей и работы с клиентами.

Основные возможности:
  • Регистрация и аутентификация пользователей
  • Разграничение прав по ролям (Администратор, Менеджер, Клиент)
  • Каталог автомобилей с поиском и фильтрацией
  • Управление заявками на аренду
  • Учёт платежей
  • Отслеживание доступности автомобилей
  • Панель администратора со статистикой
  • Адаптивный веб‑интерфейс
